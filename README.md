## Requirements:
- python3
- node 19.6.0

Step to run the project:
## Run the API
1. clone branch api first
2. create venv
```
python3.10 -m venv venv
```
3. activate venv
```
source venv/bin/activate
```
4. install requirements
```
pip install -r requirements.txt
```
5. run the api
```
python3.10 api.py
```

## Run the web
1. clone branch fe
2. run npm install
```
npm install
```
3. run the app
```
npm run
```
